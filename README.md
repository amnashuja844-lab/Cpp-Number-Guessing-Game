# C++ Number Guessing Game (CLI)

This is a command-line Number Guessing Game written in C++ as part of my
Programming Fundamentals practice.

## Concepts Used
- Random number generation
- Loops
- Conditional statements
- Input validation

## How to Run
Compile and run using a C++ compiler:

g++ main.cpp -o game
./game
